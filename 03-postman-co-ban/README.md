# Chương 3: Postman Cơ Bản

Chương này sẽ hướng dẫn bạn sử dụng Postman từ cơ bản đến nâng cao, từ việc gửi request đầu tiên đến quản lý Collections và Environments.

## Mục Tiêu Học Tập

Sau khi hoàn thành chương này, bạn sẽ:

- ✅ Thành thạo giao diện Postman
- ✅ Gửi được các loại HTTP requests
- ✅ Tổ chức requests bằng Collections
- ✅ Sử dụng Environments và Variables
- ✅ Lưu và chia sẻ work

## Nội Dung Chương

### 3.1 Cài Đặt và Giao Diện Postman
- Cài đặt Postman Desktop
- Làm quen với giao diện
- Các panels chính
- Shortcuts hữu ích

### 3.2 Gửi Request Đầu Tiên
- Tạo request mới
- Chọn HTTP method
- Nhập URL
- Gửi request và xem response
- **Thực hành**: Test với JSONPlaceholder API

### 3.3 Collections - Tổ Chức Requests
- Collection là gì?
- Tạo collection mới
- Thêm requests vào collection
- Tổ chức với folders
- Export/Import collections

### 3.4 Environments và Variables
- Environment là gì?
- Tạo và quản lý environments
- Global, Collection, Environment variables
- Sử dụng variables: `{{variable_name}}`
- **Thực hành**: Setup Dev và Prod environments

### 3.5 Request Configuration
- Query parameters
- Path variables
- Headers
- Body types (JSON, form-data, x-www-form-urlencoded)
- Pre-request Scripts

## Thời Gian Học

**Ước tính: 3-4 giờ**

## Yêu Cầu

- Đã cài đặt Postman
- Hiểu HTTP basics (Chương 2)

## Bài Tập Thực Hành

Sử dụng API: https://jsonplaceholder.typicode.com

1. Tạo collection "My First API Tests"
2. Thêm request GET all users
3. Thêm request GET user by ID
4. Tạo environment với base_url variable
5. Sử dụng variable trong tất cả requests
