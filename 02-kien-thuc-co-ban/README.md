# Chương 2: Kiến Thức Cơ Bản

Chương này sẽ cung cấp cho bạn nền tảng kiến thức về REST API, HTTP, và các khái niệm cốt lõi cần thiết để thực hiện API testing hiệu quả.

## Mục Tiêu Học Tập

Sau khi hoàn thành chương này, bạn sẽ:

- ✅ Hiểu REST API là gì và các nguyên tắc RESTful
- ✅ Nắm vững các HTTP methods (GET, POST, PUT, DELETE, PATCH)
- ✅ Phân biệt được các HTTP status codes (2xx, 3xx, 4xx, 5xx)
- ✅ Hiểu cấu trúc Request và Response
- ✅ Đọc và viết được JSON
- ✅ Phân biệt JSON và XML
